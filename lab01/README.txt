Лабораторная работа 1
Тема: Обработка внешних данных . Файлы в Си.

Задание. Разработать программу, которая обеспечивает выполнение операций над текстовыми и двоичными файлами в Си.
1.	Разработайте структуру текстового файла для представления тестовых данных варианта.
2.	В текстовом файле подготовьте тестовые данные.
3.	 Разработайте функцию чтения тестов из текстового файла и отображение их на мониторе.
4.	Разработайте структуру записи двоичного файла, для хранения в ней одного теста, указав номер теста.
5.	Разработайте функцию для вставки новых записей в двоичный файл, который будет хранить тестовые данные во время работы программы.
6.	Разработайте функцию для создания двоичного файла из записей, данные для которых хранятся в текстовом файле.
7.	Разработайте функцию, для отображения данных i- ого теста из двоичного файла.
8.	Разработайте функцию для отображения всех тестов из двоичного файла.
9.	Разработайте функции для выполнения дополнительных операций указанного варианта.
10.	Разработайте интерфейс, для выбора операций над файлом.

Вариант номер 3:
Массив из N целого типа.
N – фиксированное значение
Дополнительая операция с текстовым файлом: измените указанный тест на вновь введенные данные.
Дополнительная операция над двоичным файлом: в последнем массиве файла  найти самую длинную подпоследовательность, удовлетворяющую условию, которое определяет пользователь.
Условие определяется функцией.
